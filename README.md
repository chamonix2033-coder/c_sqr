# C-SQR イベント出欠自動取得ツール (CSQ_ListUp)

このツールは、スポーツチーム・サークル管理サービス「サークルスクエア (C-SQR)」から、指定した期間のイベント一覧とその出欠状況（出席表）を自動でスクレイピングし、CSVファイル（`events_attendance.csv`）として書き出すためのPythonスクリプトです。

## 動作環境の準備
本ツールは Python およびブラウザ自動化ライブラリである Playwright を使用しています。

### 1. 必要なライブラリのインストール
ターミナルで以下のコマンドを実行し、依存パッケージとPlaywrightのブラウザをインストールしてください。
```bash
# パッケージのインストール
pip install -r requirements.txt

# Playwright専用のブラウザ（Chromiumなど）のインストール
playwright install
```

## 実行手順

### 手順 1: 認証情報（ログイン情報）の用意
本ツールはユーザー様のログインセッション（クッキー）を利用してアクセスします。事前に `auth.json` というファイル名でPlaywrightの「Storage State (セッション情報)」を保存しておく必要があります。

*※すでにディレクトリ内に `auth.json` が配置されておりログイン状態が有効な場合は、本手順はスキップ可能です。有効期限が切れた場合や初回利用時は以下の手順で作成してください。*

#### `login_csqr.py` の使い方（auth.json の生成）
ログイン処理を自動化し、`auth.json` を生成するためのスクリプトとして `login_csqr.py` が用意されています。

1. **`.env` ファイルの作成**
   プロジェクトディレクトリにある `.env.example` を参考に、新しく `.env` ファイルを作成してください。そこへC-SQRのログインIDとパスワードを記述します。
   ```text
   CSQR_USER_ID=あなたのログインID
   CSQR_PASSWORD=あなたのパスワード
   ```
2. **ログインスクリプトの実行**
   ターミナルで以下のコマンドを実行します。
   ```bash
   .venv/bin/python login_csqr.py
   ```
3. 実行が成功すると、バックグラウンドでシステムChromeを用いて自動ログインが行われ、認証情報を含む `auth.json` が自動で生成・保存されます。

### 手順 2: 抽出条件の設定変更
`list_up_events.py` の上部（8行目〜14行目付近）にある設定項目を、取得したいデータに書き換えます。

```python
# --- list_up_events.py ---
# 設定
START_DATE = datetime(2025, 11, 1)  # 取得開始する年月
END_DATE = datetime(2026, 3, 1)     # 取得終了する年月
BASE_URL = "https://www.c-sqr.net"
OUTPUT_CSV = "events_attendance.csv" # 保存するCSVファイル名
```
必要に応じて `START_DATE` と `END_DATE` を書き換えファイルを保存します。

### 手順 3: 実行
準備ができたらターミナルで以下のコマンドを実行します。

```bash
# 仮想環境を使用している場合
.venv/bin/python list_up_events.py

# グローバル環境を使用している場合
python list_up_events.py
```

実行すると、ブラウザがバックグラウンドで起動し進捗ログがターミナルに出力されます。
1. 各月ごとのカレンダーページ (`/events/list`) を巡回し、イベントURLとカテゴリー・日時を取得します。
2. その後、各イベントごとの「出欠状況」ページ（`/attends`）を一つずつ開き、各メンバーの名前・出欠ステータス・更新日・メモ等を抽出します。

### 手順 4: 結果の確認
実行が完了すると、ディレクトリ直下に `events_attendance.csv` というCSVファイルが出力されます。

**【CSVの構成列】**
- `event_name`: イベント名（タイトル）
- `category`: カテゴリ（会議、講習会など）
- `event_datetime`: 開催日時
- `user_name`: 出席者の名前
- `status`: 出欠状況（参加、不参加、未回答、出席 等）
- `updated_at`: 出欠登録または更新された日時（代理入力などの情報含む）

このCSVファイルはExcel等で開くことが可能です。データ集計や分析にご活用ください。

## 注意事項
- **動作速度制限**: 短時間に大量のアクセスを行うとC-SQRのサーバーに負荷をかけてしまうため、イベントの詳細を取得するたびに `time.sleep(1)` で意図的に1秒の休止を入れています。外さないようお願いします。
- **レイアウトの変更**: C-SQR側のWebページのデザイン（Vue.jsのコンポーネント構造など）が将来的にアップデートされた場合、取得エラーが発生する可能性があります。その場合はスクリプトのセレクタ部分の再調整が必要になります。
